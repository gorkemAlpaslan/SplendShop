## SplendShop

SplendShop is a basic e-commerce web site built with React.js. The project features an authentication system built with Firebase, as well as a checkout system managed using local storage (note that this is currently only a frontend implementation).

## Usage

To view the live version of the website, please visit https://graceful-flan-8ed3f8.netlify.app/homepage.

## Technologies Used

I used ContextAPI, Firebase for authentication, MUI for some components and styles, some libraries for carousel and pagination, I used Local Storage to manage purchase and favorite Items
